## 🌐 Sobre o projeto

O **Site Ansiedade** é um projeto acadêmico desenvolvido com o objetivo de **informar e conscientizar sobre a ansiedade**, abordando conteúdos educativos de forma simples e acessível.

O site apresenta informações sobre sintomas, causas e formas de lidar com a ansiedade, contribuindo para a conscientização sobre saúde mental.
